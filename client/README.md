# Client

React + Vite frontend for the crypto chat UI.

## Setup

```
npm install
npm run dev
```

The UI expects the server running on `http://localhost:8080` by default.

## Usage

Try queries like:

- `price of solana`
- `market cap of doge`
- `latest ETH news`
- `is pepe a scam?`
- `trace 0x237DeE529A47750bEcdFa8A59a1D766e3e7B5F91 to 1 hop`

## Notes

- Responses include markdown plus a JSON card view.
- Sources are shown in the Sources tab.
